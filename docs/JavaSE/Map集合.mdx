# 17.3 Map集合

- Map中的数据是以键值对(key-value)的形式存储的
- key-value是以Entry类型的对象实例存在
- 可以通过key快速地查找value
- 一个映射不能包含重复的键(key是唯一的)
- 每个键最多只能映射到一个值



## HashMap

- 基于哈希表的Map接口的实现
- 允许使用null值和null键
- key值不允许重复
- HashMap中的Entry对象是无序排列的
