## main 方法

- main方法是java程序运行于Jvm的入口，具备main方法，类文件才能运行；
- 所有的Java代码严格区分大小写；
- 所有的代码的字母包括符号必须是英文输入状态下的。

```java
public class Demo{
  /*
	 * 方法名：main
	 * 方法修饰符：static
	 * 方法的返回值：void
	 * 方法的作用域：public
	 * 方法的参数：String[]--字符串类型的数组，参数类型; args--参数名称，（可以自定义）
	 * 方法功能：Java程序在jvm中的运行入口，每一个类只能有一个main方法，并不是所有的java类都有main方法
	 */
  public static void main(String[] args){

  }
}
```
