---
id: portainer-standalone
title: Docker 部署 Portainer 单机版
---

```bash
docker run -d -p 9000:9000 --name=portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer
```
