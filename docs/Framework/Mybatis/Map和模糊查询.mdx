### 模糊查询：

- 在Java代码执行时，传递通配符% %

  ```xml
  <select id="getUserLike" resultType="com.neu.pojo.User">
    select * from user where name like #{name}
  </select>
  ```

  ```java
  List<User> list = mapper.getUserLike("%李%");
  ```

  

- 在sql拼接中使用通配符

  ```xml
  <select id="getUserLike" resultType="com.neu.pojo.User">
    select * from user where name like "%"#{name}"%"
  </select>
  ```

  ```java
  List<User> list = mapper.getUserLike("李");
  ```
