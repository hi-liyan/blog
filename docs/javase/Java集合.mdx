集合属于`java.util`包下，需要import才能使用。

## 集合框架的体系结构：

- Interface Collection 包含以下三种subInterface：

  - List 序列，有序
    - 实现类：ArrayList 相当于长度可变的数组
  - Queue 队列，有序
    - 实现类：LinkedList 同时也实现了List接口，链表
  - Set 集，无序，不允许重复
    - 实现类：HashSet 哈希集

  

- Interface Map 键值对(Key:Value)

  - 实现类：HashMap 哈希表 (Key,Value)数据
