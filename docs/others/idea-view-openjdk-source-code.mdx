在查看`rt.jar`包下的`AppClassLoader`源码时，发现代码是IDEA将class文件反编译的。

记录下，下载openjdk源码，引入待IDEA中。

## 1. 源码下载

1. [下载地址](http://hg.openjdk.java.net/)
2. 选择Jdk版本

<ShadowBox img='https://upyun1.surcode.cn/imgs/20211013223606.png' />

3. 选择Jdk

<ShadowBox img='https://upyun1.surcode.cn/imgs/20211013223646.png'/>

4. 下载zip

<ShadowBox img='https://upyun1.surcode.cn/imgs/20211013223806.png'/>

## 2. IDEA引入源码

打开 IDEA 右上角`项目结构`

<ShadowBox img='https://upyun1.surcode.cn/imgs/20211013224126.png'/>

点击 `+` ,将刚刚下载好的zip压缩包添加进来

<ShadowBox img='https://upyun1.surcode.cn/imgs/20211013224356.png'/>

再次查看 `AppClassLoader`，可以显示源码了

<ShadowBox img='https://upyun1.surcode.cn/imgs/20211013224557.png'/>
