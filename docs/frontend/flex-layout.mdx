## 1. Flex 弹性布局

### 1.1 flex 概念

Flex 是 Flexible Box 的缩写，弹性盒模型。任何一个盒子都可以设置 Flex 布局。

<ImgBox src="https://images.shiguangping.com/imgs/3791e575c48b3698be6a94ae1dbff79d.png" />

```css
.box {
  display: flex
}
```

首先，要了解几个概念：

当一个盒子设置为 flex 布局后，它的子元素将成为 flex item ，子元素的排列方式会受到父元素的 flex 属性控制；

flex 布局的盒子定义了两个轴，默认是水平方向的主轴 (main axis) 和垂直方向的交叉轴 (cross axis) ，通过属性也可以设置垂直方向是主轴，水平方向是交叉轴，这个轴描述的是子元素在盒子中是横着排还是竖着排；

既然有方向就会有起点和终点，主轴上有主轴起点 (main start) 和主轴终点 (main end)，交叉轴上有交叉轴起点 (cross start) 和交叉轴终点 (cross end)，起点和终点描述的是子元素 (flex item) 在每个轴上的排列顺序，是正着排还是倒着排。知道了这几个概念可以更容易地理解 Flex 布局。

<br/>

下面就来认识下 flex 布局中的属性：


### 1.2 flex 容器属性

flex 布局的 6 个属性：

- [flex-direction](#121-flex-direction)

- [flex-flow](#123-flex-flow)

- [align-items](#125-align-items)

<br/>

#### 1.2.1 flex-direction

<RightAngleBadge>描述</RightAngleBadge>

flex-direction 属性决定主轴的方向，及 flex 盒子中子元素的排列方向，是横着排还是竖着排。

<ImgBox src="https://images.shiguangping.com/imgs/0cbe5f8268121114e87d0546e53cda6e.png" />

```css
.box {
  flex-direction: row | row-reverse | column | column-reverse;
}
```

<RightAngleBadge>属性值</RightAngleBadge>

- row（默认值）：主轴为水平方向，起点在左端；
- row-reverse：主轴为水平方向，起点在右端；
- column：主轴为垂直方向，起点在上沿；
- column-reverse：主轴为垂直方向，起点在下沿。

<br/>

#### 1.2.2 flex-wrap

<RightAngleBadge>描述</RightAngleBadge>

flex-wrap 属性描述的是，如果子元素在主轴上一行排不下是否自动换行。

<ImgBox src="https://images.shiguangping.com/imgs/903d5b7df55779c03f2687a7d4d6bcea.png" />

```css
.box{
  flex-wrap: nowrap | wrap | wrap-reverse;
}
```

<RightAngleBadge>属性值</RightAngleBadge>

- nowrap（默认）：不换行；
  <ImgBox src="https://images.shiguangping.com/imgs/9da1f23965756568b4c6ea7124db7b9a.png" />

- wrap：换行，第一行在上方（正常）；
  <ImgBox src="https://images.shiguangping.com/imgs/3c6b3c8b8fe5e26bca6fb57538cf72d9.jpg" />

- wrap-reverse：换行，第一行在下方，超出换行的部分跑到第一行。
  <ImgBox src="https://images.shiguangping.com/imgs/fb4cf2bab8b6b744b64f6d7a99cd577c.jpg" />


<br/>

#### 1.2.3 flex-flow

<RightAngleBadge>描述</RightAngleBadge>

flex-flow 是 `flex-direction` 和 `flex-wrap` 的简写形式，默认值是 `row nowrap`。

```css
.box {
  flex-flow: <flex-direction> <flex-wrap>;
}
```

<br/>

#### 1.2.4 justify-content

<RightAngleBadge>描述</RightAngleBadge>

justify-content 描述的是子元素在主轴上的对齐方式，如靠左，靠右，居中等。

```css
.box {
  justify-content: flex-start | flex-end | center | space-between | space-around;
}
```

<RightAngleBadge>属性值</RightAngleBadge>

- flex-start（默认值）：靠左对齐；
- flex-end：靠右对齐；
- center： 居中对齐；
- space-between：两端对齐，子元素之间等距；
- space-around：每个子元素两侧的间隔相同（假设靠近边框的子元素距离边框是10px，那么子元素与子元素之间的距离就是20px）。

<ImgBox src="https://images.shiguangping.com/imgs/c55dfe8e3422458b50e985552ef13ba5.png" />

<br/>

#### 1.2.5 align-items

<RightAngleBadge>描述</RightAngleBadge>

align-items 描述的是子元素在交叉轴上的对齐方式。

```css
.box {
  align-items: flex-start | flex-end | center | baseline | stretch;
}
```

<RightAngleBadge>属性值</RightAngleBadge>

- flex-start：交叉轴的起点对齐；
- flex-end：交叉轴的终点对齐；
- center：交叉轴的中点对齐；
- baseline: 项目的第一行文字的基线对齐；
- stretch（默认值）：如果项目未设置高度或设为auto，将占满整个容器的高度。

<ImgBox src="https://images.shiguangping.com/imgs/2b0c39c7e7a80d5a784c8c2ca63cde17.png" />

<br/>

#### 1.2.6 align-content

<RightAngleBadge>描述</RightAngleBadge>

align-content 属性定义了多根轴线的对齐方式。如果项目只有一根轴线，该属性不起作用。

```css
.box {
  align-content: flex-start | flex-end | center | space-between | space-around | stretch;
}
```

<RightAngleBadge>属性值</RightAngleBadge>

- flex-start：与交叉轴的起点对齐；
- flex-end：与交叉轴的终点对齐；
- center：与交叉轴的中点对齐；
- space-between：与交叉轴两端对齐，轴线之间的间隔平均分布；
- space-around：每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍；
- stretch（默认值）：轴线占满整个交叉轴。

<ImgBox src="https://images.shiguangping.com/imgs/f10918ccb8a13247c9d47715a2bd2c33.png" />

### 1.3 Flex 子元素属性

#### 1.3.1 order

<RightAngleBadge>描述</RightAngleBadge>

order 属性定义了子元素的排列顺序，数值越小，排列越靠前。默认值是 0。

<ImgBox src="https://images.shiguangping.com/imgs/59e399c72daafcfcc20ede36bf32f266.png" />

```css
.item {
  order: <integer>;
}
```

<br/>

#### 1.3.2 flex-grow

<RightAngleBadge>描述</RightAngleBadge>

flex-grow 属性定义项目的放大比例，默认值为 0，即如果存在剩余空间，也不放大。

<ImgBox src="https://images.shiguangping.com/imgs/f41c08bb35962ed79e7686f735d6cd78.png" />

```css
.item {
  flex-grow: <number>; /* default 0 */
}
```

如果所有项目的flex-grow属性都为1，则它们将等分剩余空间（如果有的话）。如果一个项目的flex-grow属性为2，其他项目都为1，则前者占据的剩余空间将比其他项多一倍。

<br/>

#### 1.3.3 flex-shrink

<RightAngleBadge>描述</RightAngleBadge>

flex-shrink 属性定义了项目的缩小比例，默认值为 1，即如果空间不足，该项目将缩小。

<ImgBox src="https://images.shiguangping.com/imgs/240d3e960043a729bb3ff5e34987904f.jpg" />

```css
.item {
  flex-shrink: <number>; /* default 1 */
}
```

如果所有项目的flex-shrink属性都为1，当空间不足时，都将等比例缩小。如果一个项目的flex-shrink属性为0，其他项目都为1，则空间不足时，前者不缩小。

负值对该属性无效。

<br/>

#### 1.3.4  flex-basis

<RightAngleBadge>描述</RightAngleBadge>

flex-basis 属性定义了在分配多余空间之前，项目占据的主轴空间 (main size)。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为 `auto`，即项目的本来大小。

```css
.item {
  flex-basis: <length> | auto; /* default auto */
}
```

它可以设为跟width或height属性一样的值（比如350px），则项目将占据固定空间。

<br/>

#### 1.3.5 flex

<RightAngleBadge>描述</RightAngleBadge>

flex 属性是 `flex-grow` / `flex-shrink` 和 `flex-basis` 的简写，默认值为`0 1 auto`。后两个属性可选。

```css
.item {
  flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]
}
```

该属性有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)。

建议优先使用这个属性，而不是单独写三个分离的属性，因为浏览器会推算相关值。

<br/>

#### 1.3.6 align-self

<RightAngleBadge>描述</RightAngleBadge>

align-self 属性允许单个项目有与其他项目不一样的对齐方式，可覆盖 align-items 属性。默认值为 `auto`，表示继承父元素的 align-items 属性，如果没有父元素，则等同于 stretch。

<ImgBox src="https://images.shiguangping.com/imgs/55b19171b8b6b9487d717bf2ecbba6de.png" />

```css
.item {
  align-self: auto | flex-start | flex-end | center | baseline | stretch;
}
```

该属性可能取6个值，除了auto，其他都与align-items属性完全一致。
