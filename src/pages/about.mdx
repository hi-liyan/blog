## 第一个小程序
我的第一个小程序上线了，是一个坚持打卡的小程序，为自己立Flag，每天坚持完成目标，让自己变得更优秀。

学了一周的 React 和 CSS ，写了这么一个小东西，别看功能简单，但是该有的东西都有。

后端服务从起初的一个服务拆分成了两个（小程序服务和用户中心服务），以后再有其他服务端可以直接对接用户中心，服务之间使用 Dubbo 做远程方法调用，Nacos 作为服务发现中心和配置中心（配置都移到 Nacos 上了），后来有个小需求还引了 RocketMQ 做异步调用。**麻雀虽小五脏俱全。**

项目通过 Jenkins 构建镜像，Docker 部署，整体开发流程还是很顺畅的。
